# AI Ticket Triage System

A desktop GUI application that automates IT helpdesk ticket triage using 
AI-driven classification. Reduces manual ticket sorting effort by 30%.

## Overview
Built as part of a hackathon project (eSMT Triage Assistant). Responsible for 
the full frontend implementation and UI/UX prototyping.

## Features
- AI-assisted ticket classification and routing
- Ticket submission and history tracking
- Error validation and input checking
- Clean desktop GUI optimized for helpdesk operators

## Tech Stack
- Python (Tkinter) — Desktop GUI
- Figma — High-fidelity UI design
- Balsamiq — Wireframe prototyping

## Project Structure
- `/designs` — Figma and Balsamiq wireframes and UI mockups
- `home.py` — Main application entry point
- `ticket_logic.py` — Ticket classification and routing logic
- `ticket_history.py` — Ticket history and tracking module

## How to Run
```bash
pip install tk
python home.py
```

## Context
Developed during a team hackathon. My role covered end-to-end frontend 
development: wireframing in Balsamiq, UI design in Figma, and implementation 
in Python/Tkinter.
